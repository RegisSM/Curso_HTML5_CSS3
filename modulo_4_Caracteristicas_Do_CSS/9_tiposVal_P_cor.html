Título da pagina

Certo desenvolvedor web ficou responsável por criar uma área de galeria de fotos no site de um cliente. Porém, o cliente pediu a ele que criasse uma camada em cima das fotos para incentivar os visitantes do site a clicar na foto para gerar eventos de click. Para cada foto ele fez da seguinte forma:
Clique aqui!
Tal desenvolvedor se utilizou da propriedade CSS [position] para alocar a imagem e a [div .thumb] no eixo Z. Ele colocou a imagem como background (fundo) e depois alocou a camada [div .thumb] em cima. Porém, na [div .thumb], ao atribuir, via CSS, uma cor e colocar a opacidade em 0.5, percebeu-se que o texto "Clique aqui" também sofria alteração de cor. Como o desenvolvedor poderia resolver esse problema na [div .thumb] deixando a cor do texto inalterada? Observação: Eixo X = Horizontal; Eixo Y = Vertical; Eixo Z = Profundidade;
